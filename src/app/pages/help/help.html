Help & User Guide

Welcome to the Bookstore Management System. This guide explains how to use each feature.

📚 View Books

Click Books on the navigation bar to display every book stored in the database.

➕ Add a Book

1. Click Add Book.

2. Enter the title, author, genre, price, stock, and cover image URL.

3. Click Add Book to save it to MongoDB Atlas.

✏ Edit a Book

Click the Edit button beside a book. Modify the information and press Save Changes.

🗑 Delete a Book

Click the Delete button beside a book. Confirm the deletion when prompted.

🔍 Search Books

Use the search box on the Books page. Books are filtered automatically while you type.

💾 Database

All information is stored in MongoDB Atlas. Changes made in the application are reflected in the database.

Troubleshooting

  • Make sure MongoDB Atlas is connected.
  • Ensure the backend server is running on port 3000.
  • Run ng serve before opening the application.
