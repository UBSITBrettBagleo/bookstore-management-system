About the System

📚 Bookstore Management System

The Bookstore Management System is a web application developed to help bookstore owners efficiently manage their inventory using modern web technologies.

Users can add, edit, delete, and view books while all information is securely stored in MongoDB Atlas.

Features

  • 📖 View all books
  • ➕ Add new books
  • ✏ Edit existing books
  • 🗑 Delete books
  • 📊 Dashboard with inventory statistics
  • ☁ MongoDB Atlas cloud database

Technologies Used

Technology Purpose
Angular Frontend Development
Node.js Backend Runtime
Express.js REST API
MongoDB Atlas Cloud Database
Mongoose Database Modeling
